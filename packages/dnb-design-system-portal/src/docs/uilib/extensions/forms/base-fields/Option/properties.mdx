## Properties

| Property   | Type                 | Description                                                                       |
| ---------- | -------------------- | --------------------------------------------------------------------------------- |
| `value`    | `string` or `number` | _(optional)_ Value for this option.                                               |
| `title`    | `React.Node`         | _(optional)_ Text title for the option.                                           |
| `text`     | `React.Node`         | _(optional)_ Secondary text.                                                      |
| `disabled` | `boolean`            | _(optional)_ Will disable the option                                              |
| `children` | `React.Node`         | _(optional)_ Optional way to provide `title`. Will be ignored if `title` is used. |
